# Task 8 - Simple Sales Dashboard Design

## Objective

Design an interactive Power BI dashboard to analyze coffee shop sales performance, identify revenue trends, and evaluate customer purchasing patterns.

## Tool Used

* Microsoft Power BI

## Dataset

* Coffee Sales Dataset (index_1.csv)

## Dashboard Components

### KPI Cards

* Total Revenue
* Total Transactions
* Average Transaction Value

### Visualizations

* Revenue Trend Over Time (Line Chart)
* Revenue by Coffee Type (Bar Chart)
* Payment Method Distribution (Donut Chart)

### Interactive Filter

* Coffee Name Slicer

## Key Insights

1. Total revenue generated during the analysis period was approximately 115.43K.
2. A total of 3,636 transactions were recorded.
3. Latte generated the highest revenue among all coffee products.
4. Card payments accounted for the majority of transactions.
5. Revenue trends varied over time, indicating changes in customer demand and purchasing behavior.

## Outcome

Developed an interactive sales dashboard that provides a clear overview of revenue performance, product popularity, transaction behavior, and payment preferences. The dashboard enables quick business insights and supports data-driven decision-making.
